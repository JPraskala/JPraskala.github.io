What is an if/else statement.

An if/else statement takes in a Boolean expression and checks if the condition is true or false. If the condition is true, then the code executes everything that is in the if block. If the condition is false, then it will execute everything in the else block.

if 5 > 3: # Execute if true else: # Execute if false

Explaining the elif statement (else if)

An if/else statement is not limited to just an if statement and an else statement. Often, programmers will include an else if statement to better control the flow of a program. In Python, the else if statement is declared as elif. An example is below.

if 5 == 3: # Will execute if true elif 5 > 3: # Will execute if the first statement is false and this statement is true else: # Will execute if both statements above are false

Keep in mind that the elif statement will only execute if the if statement is false and the elif statement is true. If the if statement is true, then it will execute the code in there and then back out of the if/else statement even if the elif statement is true.

Additional Information

Sometimes, you can have two if statements that are independent from each other. Here is an example.

if 5 > 3: # execute this code if name == 'Jake': # execute this code # These two if statements are independent from each other.
