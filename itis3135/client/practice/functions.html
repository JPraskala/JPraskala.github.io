What are Python Functions?

Python offers functions to allow a programmer to organize their program. These functions can be used to control the flow of the program and return certain values. For more information, click on this site to learn more about functions in Python.

# Each function in Python is declared with the def keyword. def example(): return 5 # This function returns the number 5. This means this function can be used in another function. def secondExample(): print(5) # This function does not return anything. This means this is a void function. A void function is a function that does not return a value. It cannot be used in an if statement. def thirdExample(parameterOne, parameterTwo): return parameterOne * parameterTwo # This function takes in two parameters meaning when the user calls this function, the user will have to give the function two arguments, or it will throw an error. def fourthExample(parameterOne: int, parameterTwo: int): return parameterOne * parameterTwo # In this example, we are stating that both parameters should be of type integer. It is not requrired to put this in, but it can be useful def fifthExample(parameterOne: int, parameterTwo: int) -> int: return parameterOne * parameterTwo # This example states which dataType we are returning in the function. Using the arrow notation, we are stating that we want to return an integer. This part is optional. def sixthExample() -> None: print(None) # If None is specified in the arrow notation, then it is a void function.
