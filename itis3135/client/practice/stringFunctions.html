What are String Functions?

String Functions allow programmers to manipulate character strings. As a reminder, a string is a collection of characters. In Python, strings can be in either single quotes or double quotes. Some string functions include upper, lower, isalpha, isdigit, etc. Click on this site to learn more about string functions.

# Suppose I have the following line below text = 'Jacob' # We can use the text variable now to call some string functions. print(text.upper()) # This will convert Jacob to Jacob print(text.lower()) # This will change the value of text to jacob print(text.isalpha()) # This will output true because all the characters are alphabets. # If there is a number or a space, then it will output false. print(text.isdigit()) # This will output false because not all the characters are numbers.
