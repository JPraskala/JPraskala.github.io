What are DataTypes?

DataTypes are known as a collection of data that the programmer can use to build programs. Some dataTypes include integers, floats/doubles, strings, booleans, and more. Which datatype to use depends on the situation. Python is a dynamically typed language meaning that if you declare a variable of one datatype, it can later be changed to another. For example, if I declare a variable called name, and it is set to a string of "Jacob", it can later be changed to a boolean or any other datatype you want. In constrast, a language like Java is a statically typed language meaning that once a variable is declared with a specific datatype, it must always be that datatype. Note: To declare comments in Python, you need to use # instead of //.

How do I declare a variable in Python?

Declaring a variable in Python is easy. You must first specify the name of the variable like variableName, and then you need to give it a value. See the example below.

a = True # Note that the T is capitalized in True. b = 'Hello' c = 1 d = 1.5

The variable a is declared as a boolean set to true. The variable b is declared as a string set to Hello (Note that strings in Python can be in either single quotes or double quotes). The variable c is declared as an integer set to 1, and the variable d is declared as a float set to 1.5.
