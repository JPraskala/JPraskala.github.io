What are loops?

Loops are a way to execute a code multiple times throughout a program. There are two different kinds of loops in Python. There is the While Loop, and there is the For Loop. Click on this site to learn more about loops in Python.

i = 0 while i < 5: print(i) # Example of a while loop. It loops while the condition is true. for i in range (0, 10): # Example of a for loop. It loops while the number is in range. In this case, it'll loop from 0 - 9.
